# 🏆 Ideal Team Lineup - FIFA World Cup 2018

## 🔍 Project Overview

This project identifies the optimal starting 11 players for the 2018 FIFA World Cup using FIFA 18 player statistics. It combines data cleaning, feature engineering, visual analysis, and player selection based on positional strengths and overall performance.

## 📁 Dataset

- Source: FIFA 18 player statistics (`CompleteDataset.csv`)
- Records: 17,981 players
- Features: 75 attributes (e.g., player name, nationality, age, skill metrics, position ratings)

## 🧠 Key Steps

1. **Data Cleaning:** Transformed wage and skill values to numeric types.
2. **Player Filtering:** Selected players based on nationality, reputation, and performance.
3. **Lineup Optimization:** Built an ideal 11-player team with best-in-role selections.
4. **Data Visualization:** Used Seaborn and Plotly to visualize player stats and trends.

## 📊 Visual Insights

- Age, wage, and skill distribution
- Country-wise skill mapping
- Positional radar charts for player comparison

## 🛠️ Tools & Libraries

- Python, Pandas, NumPy
- Seaborn, Matplotlib, Plotly
- Scikit-learn (optional), Geopy

## 📦 How to Run

1. Clone the repository
2. Place the dataset in the root directory
3. Run `Ideal_Team_Lineup_World_Cup_2018.ipynb` in Jupyter/Colab
